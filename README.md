# fleetflow-web

> **FleetFlow Production Board** · [View Kanban Board →](https://github.com/users/aldoprogrammer/projects/2)

Next.js operations and customer portal for FleetFlow.

## Stack

- Next.js · React · TanStack Query · Zustand · Tailwind CSS
- Formik · Yup · Headless UI · Floating UI · Auto Animate

## Quick start

```bash
pnpm install
pnpm run dev
```
